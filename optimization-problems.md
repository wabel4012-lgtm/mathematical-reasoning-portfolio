# Optimization Problems

## 1. Linear Optimization

Maximize:
P = 3x + 2y

Constraints:
x + y ≤ 10  
x ≥ 0, y ≥ 0

### Solution Strategy
Evaluate vertices of feasible region:
- (10,0)
- (0,10)
- (5,5)

Best result:
x = 10, y = 0 → P = 30
